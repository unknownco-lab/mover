# Mover

> Keep your system active, stay in your flow.

Mover is a productivity tool that keeps your computer active during legitimate work scenarios. Whether you're running long computations, giving presentations, or need to step away briefly without losing your session, Mover ensures your system stays active and your work uninterrupted.

Built with [Vite](https://github.com/vitejs/vite), [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/), [Electron](https://electronjs.org/), and [Tailwind CSS](https://tailwindcss.com/).

## Installation

Clone this repo and install all dependencies  
```bash
npm install
```

## Development

```bash
npm run dev
```

Or with Electron:
```bash
npm run dev:electron
```

## Build

```bash
npm run build
```

## Publish

```bash
npm run dist
```

## Features

- **Customizable Movement Patterns**: Circular, random, figure-8, horizontal, vertical, and zig-zag patterns
- **Configurable Intervals**: Set movement frequency from 1-60 seconds
- **Adjustable Distance**: Control movement distance from 10-500 pixels
- **Cross-Platform**: Works on Windows, macOS, and Linux
- **Lightweight & Efficient**: Minimal system resource usage
- **Privacy-Focused**: All processing happens locally on your machine
